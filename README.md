## 🛠️ Practice Exercises

After completing each phase, test yourself:

### Phase 1 Practice (Foundations)
1. Create a monthly budget workbook with multiple sheets
2. Use absolute and relative references correctly
3. Build formulas using all arithmetic operators
4. Calculate percentages and percentage changes
5. Format numbers as currency and percentages

### Phase 2 Practice (Core Functions)
1. Calculate total sales by region using SUMIF
2. Create a grade calculator using nested IF statements
3. Build a lookup system for product prices (VLOOKUP)
4. Extract first and last names from full names (TEXT functions)
5. Calculate age from birthdate (DATE functions)
6. Find highest and lowest values with conditions

### Phase 3 Practice (Data Management)
1. Create dropdown lists for data entry
2. Highlight overdue tasks with conditional formatting
3. Sort and filter a 100-row dataset
4. Build a Pivot Table to summarize sales by quarter
5. Create a dynamic chart from a Pivot Table

### Phase 4 Practice (Visualization)
1. Create a professional sales dashboard
2. Build a combo chart showing revenue and profit margin
3. Design a visual KPI tracker with conditional formatting
4. Create sparklines in a summary table

### Phase 5 Practice (Advanced)
1. Import data from CSV and clean it
2. Create named ranges for a financial model
3. Convert a range to a Table and use structured references
4. Write an array formula to calculate running totals
5. Use Power Query to combine multiple files
6. Record a macro to automate a repetitive task

### Phase 6 Practice (Specialized)
1. Clean a messy dataset (duplicates, blanks, errors)
2. Use Goal Seek to find break-even point
3. Create scenarios for best/worst case analysis
4. Protect sensitive data while allowing input

## ⚡ Quick Command Reference

### Essential Excel Shortcuts

**Navigation:**
```
Ctrl + Home          Go to cell A1
Ctrl + End           Go to last used cell
Ctrl + Arrow         Jump to edge of data
Ctrl + Page Up/Down  Switch between sheets
```

**Selection:**
```
Shift + Arrow        Extend selection
Ctrl + Space         Select entire column
Shift + Space        Select entire row
Ctrl + A             Select all
```

**Editing:**
```
F2                   Edit active cell
Ctrl + C             Copy
Ctrl + V             Paste
Ctrl + X             Cut
Ctrl + Z             Undo
Ctrl + Y             Redo
Delete               Clear contents
```

**Formulas:**
```
=                    Start formula
Ctrl + `             Show formulas
F4                   Toggle reference types ($)
Alt + =              AutoSum
Ctrl + Shift + Enter Array formula (legacy)
```

**Formatting:**
```
Ctrl + 1             Format cells dialog
Ctrl + B             Bold
Ctrl + I             Italic
Ctrl + U             Underline
Ctrl + Shift + $     Currency format
Ctrl + Shift + %     Percentage format
```
